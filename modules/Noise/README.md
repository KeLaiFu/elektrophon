## *noise*

<a href="https://spielhuus.github.io/elektrophon/images/adsr-panel.jpg"><img width="200" align="right" src="https://spielhuus.github.io/elektrophon/images/adsr-titel_tmb.jpg"></a>
Nothing special about this module. It creates white and pink noise and a random voltage. The noise source can be used to create percussion instruments. The random voltage source for feeding into a sample and hold module, for example the smooth and stepped generator. it does not have an inernal sample and hold like most of the random source modules to keep the module as small as possible. The base of the module is taken from Yves Usson noise generator \[[1][1]]\.
